
Steps:

1. Download the FinalSolidcoreExercise.zip and extract it
2. Open an ADMIN command prompt and navigate to the directory of the final exercise
3. Solidify the contents of the directory
4. To begin the exercise, run: runme.bat
5. If the scripts run successfully, begin troubleshooting the "McAfee Solidifier" service (scsrvc).
  a. You can start the service by running: services.msc
  b. You can also start the service by trying to run any sadmin command

Hint #1: To force start the service, you can run the command: scsrvc -d
Hint #2: Review the solidcore.log closely & use your resources!
